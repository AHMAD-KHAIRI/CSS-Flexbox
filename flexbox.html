Let's Play with Flexbox

The flex property sets how a flex item will grow or shrink to fit the space available in its flex container.
The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).
The default MAIN-AXIS for a flex container is from left to right (flex-direction: row).
The default CROSS-AXIS is from top to bottom (flex-direction: column).
The direction can be changed in the case of row-reverse or column-reverse.
The justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container.
